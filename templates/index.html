Cellframestats is a statistics/charting service intended to serve those running nodes on the 3rd generation, quantum-proof novel blockchain.

This website is built by enthusiastic Cellframe supporters who are doing this in their spare time.

If you are new to the Cellframe ecosystem, here is a quick recap of the technology:

  • Quantum hack-proof (using NIST-recommended encryption methods)
  • Supports multi-signatures
  • Designed to be modular
  • Written in the C language
  • Comes with a powerful SDK for the C language
  • Provides a Python API for plugins
