Cellframestats is a statistics / charting service intented to serve those running nodes on 3rd generation, quantum-proof novel blockchain. This website is built by some enthusiastic Cellframe supporters, who are doing this on their spare time.

If you are new to the Cellframe ecosystem, here is a quick recap of the tech:

  • Quantum hack proof (using NIST recommended encryption methods)
  • Multi-signatures
  • Written in C
  • Has a powerful SDK for C language
  • Includes also a Python API for plugins
